# Seminário PET - Introdução ao GNU/Linux

Apresentação feita em 2020 para o Programa de Educação Tutorial do curso de Ciência da Computação (PET-CC) da Universidade Federal do Rio Grande do Norte (UFRN).

## Requisitos

- node >=18.0.0

## Como executar

```sh
npm install
npm run start
```
